generator-teiron
=====================

铁人前端组的`yo generator`

使用方式

* 安装yeoman `npm install -g yo`

* 安装generator `npm install generator-teiron --save-dev`

* 到指定目录，执行`yo generator`

最终生成的目录结构如下

```
	+-- src
	|	+-- node_modules
	|	+-- css
	|	+--	images
	|	+-- js
	|	+--	sass
	|	+-- tmpl
	|	+-- package.json
	|	+-- Gruntfile.js /gulpfile.js
	|	+-- index.html
	|	+--	.gitignore
	+--	dist
	|	+--	css
	|	+--	images
	|	+--	js
	+-- .yo-rc.json
```
